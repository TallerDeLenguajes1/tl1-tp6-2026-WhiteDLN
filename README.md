
# Trabajo Practico N6





### ¿String es una tipo por valor o un tipo por referencia?


_Es un tipo por referencia._


### ¿Qué secuencias de escape tiene el tipo string?

_Las secuencias de escape son caracteres especiales que comienzan con \ (barra invertida) y representan caracteres que no se pueden escribir directamente._

| `\n` | Salto de línea |
| `\t` | Tabulación |
| `\"` | Comilla doble |
| `\\` | Barra invertida |
| `\r` | Retorno de carro |
| `\b` | Retroceso |
| `\0` | Carácter nulo (NULL) |
| `\a` | Alerta (campana) |
| `\v` | Tabulación vertical |
| `\f` | Salto de página |
| `\uXXXX` | Carácter Unicode |
| `\xXX` | Carácter Unicode hexadecimal |

### ¿Qué sucede cuando utiliza el carácter @ y $ antes de una cadena de texto?

_El prefijo @ crea una cadena textual (verbatim string) que ignora los caracteres de escape._

_El prefijo $ permite la interpolación de cadenas, insertando variables directamente dentro del texto usando { }._
